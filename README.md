# SkateCam
2d 1v1 skater game made for the 2025 tamu howdyhack

### controls:
p1:
w go
q e flip

p2:
i go
j l flip

### build from source:
prereqs: git, godot engine 4.5

steps:

Clone repo

Open in Godot

Go to Project -> Export

Install the packager for your operating system

Export

### installation
Windows:
download latest release
